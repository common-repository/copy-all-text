Just adding a copy all text button on top of the Gutenberg editor for your sanity.

== Description ==

The Copy All Text WordPress plugin adds a simple and convenient button on top of the Gutenberg editor that allows you to quickly copy all the content in the block editor. 

Whether you're trying to edit a blog post with other tools or sharing it with your team, this plugin helps you to save time and avoid frustration by making it easy to copy all the text to other places in one click.

== Installation ==

1. Install by searching "Copy All Text" via the 'Plugins' menu in WordPress dashboard or Download the plugin via WordPress.org and upload the plugin via the 'Plugins' menu in WordPress dashboard
2. Activate the plugin through the 'Plugins' menu in WordPress dashboard
3. Enjoy

== Screenshots ==

1. Copy All Text adds a button in the block editor toolbar.
2. Copy text from all blocks with just one click.
3. A notification will appear when all text has been successfully copied.

== Frequently Asked Questions ==

= Why use Copy All Text? =

Copy All Text is a WordPress plugin designed to add a "Copy All" button to the Gutenberg editor. The button allows users to quickly and easily copy all the content in the editor with a single click, saving time and effort compared to manually selecting and copying individual blocks of text.

= Is it able to copy all text to Markdown format? =

Not yet. More features are expected on future releases.

= Is it able to copy all text with images? =

Not yet. More features are expected on future releases.
